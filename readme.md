## camera_geometry

Various implementations and speed comparison of 3D-to-2D camera projection.

Including: 

- pure numpy
- opencv
- numba
- julia
- rust


Coming next:
- opencl 
- cuda
- glsl shader
- fpga

## Purpose

This is purely educational and not made to be a production-ready library.
